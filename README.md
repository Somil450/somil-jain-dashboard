🚀 Somil Jain Dashboard (PWA)

A mobile-first **Progressive Web App (PWA)** designed for productivity and daily organization.
This dashboard provides a native app-like experience with **offline support**, **installable interface**, and **local data persistence**.

---

✨ Features

✅ Task Management  
✅ Notes System  
✅ Expense / Income Tracker  
✅ Productivity Stats  
✅ Offline Capability (Service Worker)  
✅ Installable App Experience  
✅ Mobile-First UI  

---

📱 Progressive Web App Capabilities

- Installable on mobile & desktop
- Works offline
- Native app-like interface
- Fast loading via caching
- Responsive mobile UI

---

🛠️ Built With

-**HTML5**
-**CSS3**
-**JavaScript (Vanilla JS)**
-**Service Workers**
-**Web App Manifest**
-**LocalStorage**

📦 Installation (PWA)

2. Tap ⋮ menu
3. Click **Add to Home Screen**
4. Install

▶ iPhone (Safari)
1. Open the live demo link
2. Tap **Share**
3. Click **Add to Home Screen**
